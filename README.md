# 📊 Social Media Impact Analysis (2015–2060)

An end-to-end data analytics project that explores how social media usage impacts teenagers' mental health, academic performance, and digital wellbeing using Python, PostgreSQL, and Power BI.

## 🚀 Project Overview

This project analyzes a synthetic dataset containing **464K+ records** from **98 countries** spanning **2015–2060**. The goal is to extract meaningful insights through data analysis, SQL, and interactive dashboards.

## 🛠️ Tech Stack

- 🐍 Python
- 🐼 Pandas
- 📊 Matplotlib
- 📈 Statistics (Correlation, ANOVA, Chi-Square)
- 🗄️ PostgreSQL
- 📉 Power BI

## 📌 Project Workflow

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- PostgreSQL Database Design
- SQL Business Insights
- Interactive Power BI Dashboards
